# TMDB Movie App

TMDB Movie App build in Flutter.

## Features of This App:

- [x] List Trending Movies
- [x] Check Movie Details
- [x] API Integration
- [ ] Open Movie Trailer on YouTube
- [ ] Search Movies
- [ ] Navigation 2.O
- [ ] State Management
